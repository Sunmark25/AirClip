\doxysubsection{Detailed Description}
Manages user-\/related operations in the application. 

The \doxylink{class_user_manager}{User\+Manager} class provides functionalities for finding, authenticating, registering, and logging in users within the application. It interacts with a database through the \doxylink{class_database_controller}{Database\+Controller}.
